\subsection{Detailed Description}
A container for the 'ticks' used for timings

A union is used for \hyperlink{union_tick__t}{Tick\-\_\-t} so that we can get the efficiency of seperate high and low bytes, but if need be, we can have the convenience of a full 16bit word. N\-B\-: \hyperlink{union_tick__t}{Tick\-\_\-t} is intended for use with standard, equidistant, tick interrupts.
